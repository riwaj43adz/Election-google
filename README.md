1.  **Unmatched Integrity with Google Firebase**:

## 🚢 Deployment (Static & Scalable)

Thanks to the efficient static export capabilities of Vite, ElectiVate is hosted on **GitHub Pages**, but it lives in the **Google Cloud**. 

1.  **Build**: `npm run build`
2.  **Deploy**: `npm run deploy`
